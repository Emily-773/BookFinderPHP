# BookFinder (PHP Version)

A web application developed for the Advanced Web Development module.

## Features
- User authentication (login/signup using PHP)
- Google Books API integration
- Search and view books
- Save books to a personal list
- Update reading status
- Delete saved books

## Technologies Used
- PHP
- MySQL
- JavaScript
- Bootstrap
- Google Books API

## How to Run Locally
1. Install XAMPP
2. Place project in htdocs
3. Start Apache & MySQL
4. Import database
5. Visit: http://localhost/BookFinder
